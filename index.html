MathIsFun2

The opensource game is known as "Math Is Fun Too" in its full form.

My first iphone game. This askes questions about addition, subtraction, multiplication and division. The game uses teh accelerometer of the iphone to change the mode of the question. Each side is a different mode. Mode actually means one of these four, addition, subtraction, multiplication, division. Originally made in the intention to educate the pre-school kids. Looking at its potential, made opensource. The intention is to more and more mathematics game for the kids. That does mean the game can be played by elders too.

This game is made with the opensource Sparrow-Framework game engine. Which it self is a very nice and easy to use game engine.
